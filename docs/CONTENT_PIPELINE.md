# Content Pipeline — Teotl V4 · Nightmare Mode

> **Status:** Placeholder — implement during Phase 1 / Phase 4.

## Overview

```
artists/writers         tools/                  runtime
     │                     │                       │
 source assets  ──►  pack-assets.sh  ──►  assets/runtime/
 content YAML   ──►  validate-content.sh (schema check)
```

## Source Assets (`assets/source/`)

| Folder | Contents | Target format |
|---|---|---|
| `splasharts/` | Title, loading, ending artwork (PSD/PNG) | WebP / AVIF |
| `characters/` | Character sprites, portraits | WebP sprite sheets |
| `environment/` | Background tiles, parallax layers | WebP |
| `ui/` | UI chrome, icons, buttons | WebP / SVG |
| `audio/` | Music, ambience, SFX (WAV/FLAC) | OGG Vorbis / MP3 |

## Runtime Assets (`assets/runtime/`)

Optimised files served directly to the browser. Generated by `tools/pack-assets.sh`.
Should **not** be committed to source control (add to `.gitignore` once pipeline is live).

## Asset Manifest (`assets/manifest.json`)

A JSON index of all runtime assets. The TypeScript `AssetLoader` (TODO) reads
this file to pre-load required assets before the game starts.

Schema (fields):

| Field | Type | Description |
|---|---|---|
| `id` | string | Unique asset identifier |
| `path` | string | Path relative to `assets/` |
| `type` | `"image"\|"audio"\|"wasm"\|"json"` | Asset kind |
| `tags` | string[] | Optional categorisation tags |

## Content Data (`content/`)

Game data is stored as YAML for human readability and compiled/loaded at
runtime. Validation is performed by `tools/validate-content.sh` using
JSON Schema.

| Folder | Schema (TODO) | Description |
|---|---|---|
| `world/` | `schemas/location.json` | Location nodes and connections |
| `characters/` | `schemas/character.json` | NPC definitions |
| `items/` | `schemas/item.json` | Item registry |
| `quests/` | `schemas/quest.json` | Quest definitions |

## Localisation

All user-facing strings in YAML content should use a `_i18n` key pattern.
Full localisation support is planned for Phase 5.
